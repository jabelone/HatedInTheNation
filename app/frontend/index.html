About Hated in The Nation

This was created for QUT's CAB432 unit. It parses a feed of tweets in near real time and uses NLTK to run sentiment analysis on them. It then ranks all of the tags (@ or #) in order of lowest to highest sentiment. This demo uses a list of Australian policitians to demonstrate.

The infrastructure this site is running on is designed to scale with the demand, as this was part of the assignment. Under heavy load, the load balancer automatically launches more instances. Under lighter load, it termiante them to save on costs/resources. If there is more than one instance running, the load balancer will distribute requests between all of the instances.

A couple of important things to note. This tool is in absolutely no way accurate. Twitter is a weird ecosystem of very vocal people from both sides of the political spectrum. It is however, interesting nonetheless. :)
